# NestJS-Server-App

A development practice of server application development using NestJS, JWT, TypeORM, Jest and PostgreSQL.

## Getting Started

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```
